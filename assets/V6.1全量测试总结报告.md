## 六、急性vs慢性双轨制关键验证

V6.1新增R010吸收兜底协议的核心验证点：同一规则（R010），在慢性（>2周）和急性（<2周）两种场景下，系统输出是否一致且正确。

| 对比维度 | Q2（慢性路径） | Q6（急性路径） |
|---------|--------------|--------------|
| **场景** | 长期消化不好·想换水解蛋白粉 | 拉肚子1周·加补剂后出现·想换水解蛋白粉 |
| **R010触发** | ✅触发（慢性>2周） | ❌未触发（急性<2周） |
| **触发判断** | 精准 | 精准 |
| **路由** | R010触发→吸收兜底协议慢性路径 | R010未触发→路由B→吸收兜底协议急性路径 |
| **兜底输出** | 建议就医+营养调整方案 | 7天症状观察+预设就医路径 |
| **医学兜底** | "建议去医院看看肠胃" | "如果7天后拉肚子还没好，建议去医院看看" |
| **风格一致性** | ✅ | ✅ |
| **CM002匹配** | 高（长期忽视吸收通道） | 中（急性因果关联明确，非典型模式） |

> **结论：双轨制验证通过。** R010的2周阈值判断准确，急性/慢性两条路径输出风格一致、医学兜底一致、用户语言一致。CM002匹配度从"高"到"中"的诚实标注体现了系统对两种场景差异的准确识别。

每天量一次腰围（早晨空腹，肚脐水平绕一圈），用数据说话，别凭感觉——"感觉粗了一点"经常不准
每天早晨空腹称体重，连称7天，你会发现前几天涨得快，后面会慢下来甚至小幅回落，这就是水分在稳定

**为什么**

你从一碗饭加到两碗，还加了面包燕麦——碳水吃多了，身体会把多余的碳水存成糖原，而每存1克糖原就会绑定约3克水分。你涨的这2kg，大部分是糖原+水，不是脂肪。你怕的胰岛素也不是敌人，它只是身体收到"吃东西了"之后发出的信号，告诉细胞"有原料来了，开始干活"——增肌期你需要这个信号。

**每天记录什么**

- 早晨空腹体重
- 早晨空腹腰围
- 今天碳水大概吃了多少（不用精确，大概对比上周：差不多/更多/更少）
- 有没有腹胀或消化不适

**7天后会怎样**

| 如果出现这个情况 | 说明什么 | 下一步怎么做 |
|---|---|---|
| 体重稳定或小幅回落，腰围没变 | 确认是水分波动，不是脂肪 | 维持饭量，等体重稳定后再评估 |
| 体重还在快速涨，腰围也明显变大 | 可能热量加得有点多了 | 碳水不用减，面包/燕麦稍微控制量 |
| 体重稳定了但腰围没变，精力也不错 | 一切正常，增肌在轨道上 | 继续保持，关注训练表现有没有提升 |
| 出现腹胀或消化不适 | 吃太多身体处理不了 | 把加量的部分拆成更多顿，每顿少吃一点 |

### Q6 — 吸收维度·急性vs慢性+医学兜底一致性压测

#### 测试问题

> 我172cm 59kg，增肌三个月，最近一周开始拉肚子，之前肠胃一直挺好的。上周开始加了蛋白粉和增肌粉，每天两勺。饮食其他没变，还是4顿正餐。睡眠7小时左右，训练3天/周。拉肚子是不是说明我吸收不了？要不要停掉补剂？还是说换个水解蛋白粉？

**压测点：** R010是否触发（急性<2周→观察 vs 慢性→就医）/ 医学兜底建议是否出现且与Q2一致 / K库触发词映射（拉肚子→K010）/ 用户层语气是否为对话

**得分：95/100**（V6无对应场景，新增能力）

#### 教练回答（用户层）

你现在最大的问题是：你同时加了两种新东西进身体，肠胃在抗议。之前肠胃一直没问题，加了蛋白粉和增肌粉就开始拉肚子——这不是你"吸收不了"，是你的消化系统突然收到一大堆新东西，处理不了。换水解蛋白粉之前，得先搞清楚到底是哪个在捣乱，还是两个都有问题。不搞清楚就换，大概率白换。

**这7天你要做什么**

先把蛋白粉和增肌粉都停了，不是永久停，是先让肠胃恢复。等你不再拉肚子了，再一样一样加回来试
4顿正餐照常吃，你现在不用补剂光靠正餐也不会掉肌肉，别慌
停补剂后，只选一种先加回来——建议先试蛋白粉，从半勺开始，吃3天没问题再加到一勺；蛋白粉OK了，再试增肌粉，也是从半勺开始。别两个同时加
如果7天后拉肚子还没好，建议去医院看看，别自己扛着。

**为什么**

吃进去的东西要先被消化分解，身体才能用得上。你现在拉肚子，说明食物直接穿肠而过，根本没来得及吸收。你上周同时加了蛋白粉和增肌粉——两样都是新东西，身体一下子不适应很正常。就好比你一直走一条熟悉的路，突然让你同时走两条新路，迷路不奇怪。先把路走通一条，再走另一条。

**每天记录什么**

- 大便情况（正常/偏稀/拉肚子）
- 停补剂后第几天
- 有没有腹胀或其他不适
- 体重（早晨空腹）
- 如果开始试补剂：试的哪种、多少量、反应怎样

**7天后会怎样**

| 如果出现这个情况 | 说明什么 | 下一步怎么做 |
|---|---|---|
| 停补剂后2-3天就不拉了 | 确认是补剂引起的急性反应 | 按计划一样一样试，先蛋白粉半勺起 |
| 试蛋白粉半勺没问题，一勺也没事 | 蛋白粉OK，问题可能在增肌粉 | 再试增肌粉半勺起，观察反应 |
| 试增肌粉半勺又拉了 | 增肌粉是你的触发点 | 停增肌粉，蛋白粉继续，碳水从食物补 |
| 两种半勺都没问题 | 之前是一下加太多，身体适应不了 | 慢慢加量，每次只加一样，隔3天再下一步 |
| 停补剂7天还拉肚子 | 可能不是补剂的问题 | 建议去医院检查一下肠胃 |

#### 教练回答（审计层）

**【问题分类】** 吸收维度（A）— 用户新增补剂后出现急性腹泻（1周），疑似补剂引发的消化不适
**【规则扫描】** R010 ❌未触发（拉肚子1周<2周阈值，属急性）。R001-R009全部未触发或缺数据。
**【路由判断】** 数据充足但R库无覆盖规则 → 路由B → 吸收兜底子协议（急性路径）
**【决策来源】** [知识推理-非规则]　**【主因】** 疑似方向：新增补剂引发的急性消化反应

**【C库】** CM002匹配度"中"——诚实区分急性排异反应vs慢性通道失败
**【P库】** P006/P007 全部通过
